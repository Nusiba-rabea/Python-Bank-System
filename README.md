# Python Bank Account Management System

## Description
This project is a simple Bank Account Management System built using Python basics only.

The program allows users to create accounts, login, manage their balance, transfer money, and change their passwords.

## Features

### Main Menu:
- Register a new account
- Login to an existing account
- Exit the program

### Bank Menu:
After successful login, users can:
- Check account balance
- Deposit money
- Withdraw money
- Transfer money to another user
- Change password
- Logout

## Technologies Used
- Python
- Variables
- Input and Output
- Conditions (if / elif / else)
- Loops (while)
- Functions
- Dictionaries

## How to Run

Run the main file:


python system_bank.py


## Validation Rules
- Username cannot be empty.
- Username must be unique.
- Password must contain at least 6 characters.
- Deposit and withdrawal amounts must be greater than zero.
- Transfer requires an existing user and sufficient balance.
